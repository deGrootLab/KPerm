Installation¶

pip¶

You can install the latest release of KPerm using pip:

pip install kperm

Development¶

To use the development (trunk) version of KPerm, download from GitHub and install it via pip:

git clone https://github.com/deGrootLab/KPerm
cd KPerm
pip install -e .
